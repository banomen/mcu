# Библиотека libopencm3 в проектах под управлением cmake

Набор сценариев cmake, обеспечивающий интеграцию библиотеки libopencm3.
Подключение библиотки реализуется за счет функции fetch_content.

Предполагается, что библиотека включается в основной проект за счет функции add_subdirectory.

На этапе конфигурирования проекта извлекается информация об используемом устройстве (переменная DEVICE основного сценария проекта) и выполняется сборка требуемого варианта библиотеки.
Сборка предназначена для статической компоновки.
Дополнительно, из базы данных устройств извлекаются параметры устройства, используемые затем в качестве значений макросов.
